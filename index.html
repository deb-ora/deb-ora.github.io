Porque escolher esta Framework?

Confira os principais motivos para começar a usar Angular

Angular

Angular é um framework para desenvolver aplicações em diversas plataformas, mantido e desenvolvido pela Google. Seu propósito é ser um framework SPA (single page applications). Foi criado para gerenciar totalmente a página. Ele faz tudo, gerencia componentes, roteamento, requests e tudo mais. Suas aplicações são monolíticas.

Testes

Use Karma para testes unitarios e cada vez que você salvar saiba se você estragou algo. E Protractor faz seus cenários testes rodar mais rápido e estavel.

Construção de Templates

A construção de templates com Angular é fácil, os templates são arquivos HTML comuns. As diretivas de templates do angular como if , for, integração com variáveis, são todas simples e intuitivas.

Integração com Lógica

Cada componente tem uma classe as quais as funções são visíveis aos templates. Basta criar uma função com a lógica desejada e utiliza-la.

Gerenciador de Formularios

Angular permite gerenciar formulários de 3 formas: NgForm, Reactive Forms e atribuindo variáveis aos inputs que é o jeito mais simples e fácil.

Comunicação Cliente Servidor

Angular 4.3 possui um componente chamado HttpClient que concentra a lógica de comunicação com servidor via HTTP e faz todo o trabalho sujo de gerenciar requisições e suas respostas, deixando o desenvolvedor bem tranquilo

Componente Router

Apps Angular carregam rápido com o novo Componente Router, que separa automaticamente e entrega ao usuário somente o código para renderizar o view requisitado.

Angular pertence à stack MEAN, sigla que descreve às quatro tecnologias que cobrem os principais aspectos de desenvolvimento de produtos de software.
  • M - MongoDB, NoSQL database
  • E – Express, back-end middleware
  • A – Angular, front-end framework
  • N – Node.js, runtime environment.
